# Habit Tracker with Streak - Web Engineering I

## Description
This project is a Habit Tracker web application developed for the Web Engineering I course. It helps users build and maintain daily habits by tracking progress and maintaining streaks.

## Features
- Add daily habits
- Mark habits as completed
- Track daily/weekly streaks
- Simple and user-friendly interface
- Progress tracking system

## Technologies Used
- HTML
- CSS
- JavaScript 
- Frontend Web Development Concepts

## Concepts Covered
- Web page structuring
- User interface design
- Event handling
- Basic state management
- Streak tracking logic

## Purpose
To help users develop consistency in habits while learning core web engineering concepts and frontend development skills.
